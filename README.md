# Tombala Matik 

Tombala Matik is a fun SwiftUI-based app for drawing bingo numbers from 1 to 90. It includes auto-draw functionality, adjustable draw intervals, sound support, and a beautiful user interface with a splash screen and iPad layout optimization.

## ✨ Features
- Auto-draw with timer
- Draw interval customization
- Sound support with AVSpeechSynthesizer
- Splash screen and LaunchScreen
- iPad responsive design
- Localization support (en-tr)


## 🚀 Getting Started
Just clone the repo and open `BingoDrawSwiftUI.xcodeproj` in Xcode.
